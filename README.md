## 🖥️ 網站功能說明

### 學生頁（index.html）
- 填寫姓名、班級、服務組別、電子郵件、心得（最多 500 字）
- 心得牆即時顯示，可依環境組 / 聊天組篩選
- 老師評語以**紅色字體**顯示，並蓋上「盧盧印章」

### 教師後台（teacher.html）
- 密碼登入，檢視所有學生心得
- 可依狀態、組別、關鍵字篩選
- 填寫評語後自動通知學生（含學生原始心得＋老師評語）

## ⚙️ 技術架構

| 功能 | 技術 |
|------|------|
| 資料儲存 | Firebase Firestore（需填入 config） |
| 電子郵件通知 | EmailJS（免費 200 封／月，需填入 config） |
| 網站部署 | GitHub Pages |
| 自動化測試 | Jest + jsdom |

### 執行測試
```bash
cd tests
npm install
npm test
```
